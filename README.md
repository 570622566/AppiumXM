# AppiumXM
基于 Appium + testNG + java + maven 适用于Android、iOS自动化测试框架，Excel关键字驱动无需编写代码，支持jenkins持续集成，Android真机或模拟器多设备并发执行以及性能监控，iOS只支持基本操作
# 开发配置
Appium  1.6.3<p>
maven   3.3.9<p>
OSX  10.11.6<p>
Xcode  8.2.1<p>
window 10
